NVM的安装和使用

Sunday2023-01-29 20:24:50NVMNVM

nvm是一个node的版本管理工具，可以很方便地切换node版本，以满足不同的需要

nvm安装教程open in new window

# nvm安装

下载地址：nvm 1.1.9open in new window

下载完成后，运行安装程序，选择好nvm的安装目录和nodejs的安装目录后，点击安装即可

# 环境变量配置

nvm在安装成功后，一般会自动帮我们配置好环境变量（在PATH路径下添加NVM_HOME和NVM_SYMLINK，NVM_SYMLINK选择nodejs目录的路径）

# npm镜像源更改

修改为淘宝镜像源后，可提升npm下载速度

在nvm安装目录中setting.txt末尾追加以下内容

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

# nvm的用法

nvm list available      #查看可用的node版本
nvm install 16.14.0     #安装指定版本的node
nvm use 16.14.0         #使用指定版本node
nvm list                #查看已安装的node版本
npm install -g hexo-cli #全局安装Hexo，安装后可以使用一系列hexo命令

# nvm命令补充

nvm list    //查看已安装的nodejs版本
nvm on      // 启用node.js版本管理
nvm off     // 禁用node.js版本管理(不卸载任何东西)
nvm install 16.14.0      // 安装指定版本的nodejs
nvm use 16.14.0      //使用指定版本的nodejs
nvm uninstall 16.14.0   // 卸载指定版本的nodejs
